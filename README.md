# PropTech Application

## Technologies Used

- Reactjs
- Nodejs
- Expressjs
- Mongodb

### Setting up the project

Fronted

```node
cd fronted
npm install
```

Backend

```node
cd backend
npm install
```

AI Servies

```node
cd ml-services
pip install -r requriements.txt
```

backend/.env


ml-services/.env

```

```
